Detailed Description

Mixed constraint class. Depending on the parameter 'isInequalityConstraint' during the construction it can be an Equality constraints ( \(Ex + Gu = f\) or \(EX + GU = f\))
or an Inequality constraints ( \(Ex + Gu\leq f\) or \(EX + GU\leq f\))
with \(X=[x_1^T ... x_N^T]^T\) and \(U=[u_0^T ... u_{N-1}^T]^T\).

Constructor of the control constraint. Create a constraint of type \(Ex + Gu\leq f\) or \(Ex + Gu = f\) or \(EX + GU = f\) or \(EX + GU\leq f\)
with \(X=[x_1^T ... x_N^T]^T\) and \(U=[u_0^T ... u_{N-1}^T]^T\).
As \(U\) is the optimization variable, \(Ex + Gu\leq f\) or \(Ex + Gu = f\) is transformed to be \(AU\leq b\) or \(AU = b\). Perform a move semantic if an rvalue is given (this is faster).
