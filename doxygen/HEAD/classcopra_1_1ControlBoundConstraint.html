Detailed Description

Control Bound constraint. It bounds the optimization \(\underline{u}\leq u \leq\overline{u}\).

Constructor of the control bound constraint. Create a constraint of type \(\underline{u}\leq u\leq\overline{u}\) or \(\underline{U}\leq U\leq\overline{U}\) with \(U=[u_0^T ... u_{N-1}^T]^T\).
As \(U\) is the optimization variable, \(\underline{u}\leq u\leq\overline{u}\) is transformed to be \(\underline{U}\leq U\leq\overline{U}\). Perform a move semantic if an rvalue is given (this is faster).

Parameters
lowerThe lower bound \(\underline{u}\) of the constraint
upperThe upper bound \(\overline{u}\) of the constraint
Exceptions
Throwan std::domain_error if lower and upper are not of the same dimension
