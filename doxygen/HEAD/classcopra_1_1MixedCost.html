Detailed Description

Mixed cost function class. This cost function looks for a minimization of a linear combination of trajectory and control. Mathematically, it is \((MX+NU-p)^TW_M(MX+NU-p) \Leftrightarrow sum_k w_m\|Mx_k+Nu_k-p\|^2\).
