Detailed Description

Abstract base class that represents cost functions. Any derived class of this one can be added to the MPC.

See also
MPC::addCost. A cost function is written \(\frac{1}{2} x^T Q x + c^T x\). Or is written by the extended version \(\frac{1}{2} x^T \tilde{Q} x + \tilde{c}^T x\) with \(\tilde{Q}=[RR^T+EQ^{\minus 1}E^T, E; E^T, Q]\) and \(\tilde{c}=[r; f]\)
