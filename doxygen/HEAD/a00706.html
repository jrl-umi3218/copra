Detailed Description

Structure representing all variables of a system for performing preview control. Such system is defined as follow: \(x_{k+1} = Ax_{k} + Bu_{k} + d\). After performing a recursion, this system can be represented in as: \(X = \Phi x_{0} + \Psi U + \Xi\).

Note
\(X = [x_0^T x_1^T ... x_N^T]^T\) and \(U = [u_0^T u_1^T ... u_{N-1}^T]^T\) where \(N\) is the dimension of the system (the number of steps).
So, \(X\) dimension is \(N+1\) and \(U\) dimension is \(N\).
