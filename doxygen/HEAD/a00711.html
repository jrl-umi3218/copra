Detailed Description

An interface to the quadratic solvers.

This interface is more like a pseudo-interface (it is not an abstract class). This class allows to have a base class for all solvers. It provides all the necessary functions for using a qp solver. In case of a QP does not have a corresponding function it sends a warning.

Solve the problem.

Solve the system:

\[ \left\{ \begin{array}{cl} \min\limits_x & \frac{1}{2} x^T Q x + c^T x \\ & A_{eq} x = b_{eq} \\ & A_{ineq} x \leq b_{ineq} \\ & XL \leq x \leq XU \end{array} \right. \]

Parameters
QAn N-by-N symmetric positive definite dense matrix.
cAn N-by-1 dense vector.
AeqEquality constrain. An M-by-N dense matrix.
beqEquality constrain. An M-by-1 dense vector.
AineqInequality constrain. An P-by-N dense matrix.
bineqInequality constrain. An P-by-1 dense vector.
XLLower bound of the decision vector. An N-by-1 dense vector.
XUUpper bound of the decision vector. An N-by-1 dense vector.
Returns
The qp solver result.

Implemented in copra::OSQPSolver, copra::LSSOLSolver, copra::QLDSolver, copra::QuadProgDenseSolver, and copra::GUROBISolver.
