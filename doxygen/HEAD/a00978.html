int copra::QLDSolver::SI_fail ( ) const
overridevirtual

Get information of eventual fail's solver output as define by the solver documentation.

Returns
0 The optimality conditions are satisfied
1 The algorithm has been stopped after too many iterations
2 Termination accuracy insufficient to satisfy convergence criterion
3 Internal inconsistency of QL, division by zero
4 Numerical instability prevents successful termination
5 Length of a working array is too short
>100 Constraints are inconsistent and fail=100+ICON, where ICON denotes a constraint causing the conflict

Solve the problem. Solve the system:

\[ \left\{ \begin{array}{cl} \min\limits_x & \frac{1}{2} x^T Q x + c^T x \\ & A_{eq} x = b_{eq} \\ & A_{ineq} x \leq b_{ineq} \\ & XL \leq x \leq XU \end{array} \right. \]

Parameters
QAn N-by-N symmetric positive definite dense matrix
cAn N-by-1 dense vector
AeqEquality constrain. An M-by-N dense matrix
beqEquality constrain. An M-by-1 dense vector
AineqInequality constrain. An P-by-N dense matrix
bineqInequality constrain. An P-by-1 dense vector
XLLower bound of the decision vector. An N-by-1 dense vector
XUUpper bound of the decision vector. An N-by-1 dense vector
Returns
The qp solver result
